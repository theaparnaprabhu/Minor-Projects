# Visuals

This folder contains all visual outputs related to the RC Circuit Simulator project.

## Contents

- `RC_Plot.png`  
  Charging and discharging curves generated using MATLAB

- `LTspice_Simulation.png`  
  Transient response waveform from LTspice simulation

- `LTspice_circuit.png`  
  RC circuit schematic built in LTspice

- `RC_UI.png`  
  Screenshot of the MATLAB App Designer interface

---

## Purpose

These visuals demonstrate:
- Theoretical behavior of RC circuits
- Comparison between MATLAB and LTspice simulations
- Functionality of the interactive application
